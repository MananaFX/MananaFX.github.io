

{% poem 民国结婚证, %}

两姓联姻，一堂缔约

良缘永结，匹配同称。

看此日桃花灼灼，宜室宜家

卜他年瓜瓞绵绵，尔昌尔炽

谨以白头之约，书向鸿笺

好将红叶之盟，载明鸳谱

此证。

{% endpoem %}
